Examen Git y Github

Alumn@: Joaquin Barrero Rodriguez

  1. Que es un sistema de control de versiones?

    Es una herramienta que ayuda a tener un control de los cambios en el codigo fuente a lo largo del tiempo. Podemos guardar fotos o imagenes del estado del codigo lo que ayuda a poder revisar comprar o incluso retroceder a versiones anteriores.

  2. Cual es la diferencia entre git y github?

    Git es un sistema de contro de versiones y Github es una plataforma donde se puede subir el codigo y utiliza Git para hacer el conbtrol de versiones.

  3. Cómo creo una rama en git?

    Cualquier modulo pueder bifurcarse (o crear rama) con el el comando git branch 'nombre_rama'

  4. Cómo clono un repositorio?

    Utilizando el comando git clone 'url_donde se aloja' hariamos una copia completa de un repositorio remoto en nuestra maquina local

  5. Cómo puedo bajar los ultimos cambios de un repositorio?

    utilizando el comando git pull

  6. Cómo puedo gener un tag?

    usando el comando git tag -a 'nombre_tag' -m '"Descripcion"'

  7. Cómo hago una mezcla de ramas?

    utilizando el comando git merge

  8. Cómo puedo moverme a un commit especifico?

    utilizando el comando git checkout 'nombredeltag' o el numero de commit que podemos ver con el comando git log --oneline --decorate

  9. Cómo puedo borrar una rama?

    git branch -d 'nombre_rama'

  10. Cómo puedo borrar un tag?

    [git tag -d 'nombre_etiqueta'

  11. Cómo puedo ver los cambios que estan por enviarse al repositorio?

    git status

  12. Para que sirve el comando git remote -v?

    git remote sirve para configurar los repositorios que sigues. La opción -v especifica el argumento "verbose" para mostrar la url del repositorio después del nombre de dicho repositorio: "git remote [-v | --verbose]"

  13. Cómo le digo a git cual es mi email y correo?

    git config --global user.name "Tu Nombre" git config --global user.email "tuemail@example.com"

  14. Cuales son los pasos para conectar git con github?

    Despues de hacer la gestión de credenciales (bien por SSH o por token) git remote add origin https://github.com/tu-usuario/tu-repositorio.git git config --list

  15. Cómo puedo crear un alias en git?

    git config --global alias.'nombre_alias' "comando a ejecutar" git 'nombre_alias' lo ejecutaria
